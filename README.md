# Super-Light Web WhatsApp API Server

Ce projet est structuré en tant que monorépo. Le code source principal, incluant le backend et le frontend, se trouve dans le dossier `Super-Light-Web-WhatsApp-API-Server-main`.

## Déploiement Rapide (Docker)

1. Accédez au dossier du projet :
   ```bash
   cd Super-Light-Web-WhatsApp-API-Server-main
   ```

2. Suivez les instructions du [README.md](./Super-Light-Web-WhatsApp-API-Server-main/README.md) pour configurer les variables d'environnement et lancer Docker.

## Structure du projet

- `Super-Light-Web-WhatsApp-API-Server-main/` : Backend Express.js
- `Super-Light-Web-WhatsApp-API-Server-main/frontend/` : Frontend Next.js
